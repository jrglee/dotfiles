-----------------

* git
* git-extras
* bash
* bash-completion
* bash-git-prompt
* vim
* macvim
* docker
* docker-compose
* docker-machine
* hg (for other dev tools)

Setup bash
----------

Change OSX default shell to use the one installed by Homebrew

    chsh -s /usr/local/bin/bash

Install dotfiles

    ~/dotfiles/install.sh
